# Olive UI Clone

This project is a pixel-accurate frontend recreation of a landing page from the Olive App website, built as part of a frontend assessment.

 Live Demo

## Overview

The goal of this project was to translate a static UI screenshot into a responsive and functional frontend using modern web development tools.

The implementation focuses on:
- Clean layout structure
- Accurate typography and spacing
- Component-based design using React
- Visual similarity to the original design


##  Tech Stack

- React.js
- CSS3 (Flexbox)
- JavaScript 
- Vercel (Deployment)
- Git & GitHub


## Features

- Responsive Navbar with dropdown indicators
- Hero section with:
  - Trusted users (avatar group)
  - Headline + description
  - CTA button + link
- Phone mockup section displaying UI
- Clean and modern layout
